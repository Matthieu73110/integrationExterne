# Étapes pour Lancer l'Application
Pour démarrer l'application, suivez ces étapes dans chaque partie du projet (`auth-service`, `core-service`, et `pdf-service`):

1. Installer les dépendances :
   ```bash
   npm i
   ```
2. Lancer l'application :
   ```bash
   node app.js
   ```
